# OpenClaw 中文本地化最佳实践

## 1. 语言与区域配置
### 1.1 全局默认设置
- 默认语言：简体中文（zh-CN）
- 时区：Asia/Shanghai（GMT+8）
- 日期格式：YYYY年MM月DD日
- 时间格式：HH:mm:ss
- 货币单位：人民币（CNY）
- 数字格式：使用中文千位分隔符（12,345,678）

### 1.2 响应规范
- 用户使用中文提问时，必须使用中文回答
- 避免使用英文术语缩写，首次出现时需注明中文含义
  示例：API（应用程序编程接口）、SDK（软件开发工具包）
- 技术文档和帮助信息优先提供中文版本
- 口语化表达符合中文用户习惯，避免生硬的机器翻译

## 2. 内容适配
### 2.1 文化适配
- 日期表达符合中国习惯：年-月-日，周几/星期几
- 节假日识别：支持中国法定节假日、二十四节气、传统节日
- 单位换算优先使用公制单位（米、千克、摄氏度等）
- 避免使用不符合中国法律法规和公序良俗的内容

### 2.2 功能适配
- 地址识别：支持中国省市区三级地址格式
- 手机号格式：11位手机号，支持+86前缀
- 身份证号：支持18位中国居民身份证号格式验证
- 银行卡号：支持国内主流银行卡号格式识别

### 2.3 服务适配
- 天气查询默认使用中国气象局数据源
- 地图服务优先使用高德/百度地图API
- 快递查询支持国内主流快递公司
- 日历支持农历、节假日调休显示

## 3. 技能本地化
### 3.1 飞书生态
- 所有飞书相关技能默认使用中文界面和操作提示
- 文档模板符合国内企业办公习惯
- 权限管理适配飞书组织架构体系
- 通知消息使用中文企业沟通常用语

### 3.2 商业技能
- 电商类技能默认支持人民币结算、国内物流
- 财务类技能适配中国会计准则、税务政策
- 项目管理类技能支持敏捷开发、OKR等国内常用方法论
- 客户服务类技能符合国内用户沟通习惯

## 4. 性能优化
- 中文分词使用国内开源分词库（jieba、THULAC等）
- 语义理解模型优先训练中文语料
- 响应生成优化中文语序和表达方式
- 长文本摘要符合中文阅读习惯

## 5. 合规要求
- 所有输出内容符合《网络安全法》、《数据安全法》等相关法律法规
- 敏感内容过滤符合国内监管要求
- 用户数据存储在国内服务器，符合数据出境管理规定
- 提供内容审核和举报机制
