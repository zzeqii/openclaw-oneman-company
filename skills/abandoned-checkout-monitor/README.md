# abandoned-checkout-monitor

Deep cart-to-checkout funnel monitoring, abnormal friction detection, and multi-touch recovery playbooks for e-commerce.

## Usage

Use when:
- Tracking checkout conversion
- Detecting abnormal drop-off points
- Generating recovery email sequences for abandoned carts
- Analyzing why checkout isn't converting

## Token Saving

This skill analyzes locally via CLI, only summary stats are sent to chat - keeps token usage low.
