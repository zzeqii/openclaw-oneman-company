# deadline-monitor

Deadline tracking and automatic reminder system for all projects.

## Features

- Tracks all project deadlines
- Automatic graduated reminders: 24h → 12h → 1h → 10m before deadline
- Sends reminders to Feishu automatically
- Detects over due tasks and escalates alerts

## Token Saving

All deadline checking runs locally, only alert messages sent - very low token usage.
