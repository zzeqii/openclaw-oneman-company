# deadline-monitor
> **截止时间监督告警复盘skill**，保证所有任务按时交付，延期自动告警复盘，是one-man-company按时交付的核心保障机制。

## 核心能力
- **任务截止时间绑定**：每个任务创建时绑定明确截止时间，自动计时
- **自动告警提醒**：距离截止时间剩余1小时提醒一次，超时后每分钟提醒一次，直到任务完成
- **延期复盘机制**：超时后自动触发复盘，分析延期原因，更新流程规则避免再犯
- **质量时间双保证**：既要保证时间，也要保证质量，时间到了必须交付，质量不达标不算完成
- **自动统计**：自动统计任务完成率，延期率，持续优化流程

## 工作流程
```
1. 创建任务 → 绑定截止时间 + 质量要求
2. 自动计时 → 距离截止剩余1小时自动提醒
3. 到期未完成 → 每分钟告警提醒 + 触发复盘 → 分析原因 → 更新机制
4. 任务完成 → 记录完成时间，统计按时/延期
```

## 告警规则
- 剩余1小时 → 推送一次提醒
- 已超时 → 每分钟推送一次，直到完成
- 超时超过1小时 → 触发复盘，分析原因，更新机制

## Usage
全自动运行，不需要人工干预，创建任务时绑定截止时间，之后自动监控告警。

## 适配one-man-company
- 保证所有任务按时交付，不延期
- 自动复盘持续优化流程，越用越快
