## 🏗️ 架构设计

### 1. 核心类结构
```
AgentBrowser
├── 属性
│   ├── browser: Browser 实例
│   ├── page: Page 实例
│   └── downloadPath: 下载目录
├── 核心方法
│   ├── init(): 初始化浏览器
│   ├── search(): 多引擎搜索
│   ├── navigate(): 页面导航
│   ├── click(): 点击元素
│   ├── fill(): 填写表单
│   ├── extractContent(): 提取结构化内容
│   ├── screenshot(): 保存截图
│   ├── savePage(): 保存HTML
│   └── close(): 关闭浏览器
└── 扩展方法
    ├── scrollToBottom(): 滚动到底部
    ├── extractTable(): 提取表格
    ├── waitForSelector(): 等待元素
    └── evaluate(): 执行自定义JS
```

### 2. 工作流设计
```
搜索工作流：初始化浏览器 → 多引擎搜索 → 结果去重 → 批量爬取 → 内容提取 → 结构化导出
知识库学习流：初始化浏览器 → 访问文档首页 → 提取导航链接 → 批量爬取 → 内容整理 → Markdown导出
```

## 🚀 部署说明

### 1. 环境依赖
```bash
# 安装依赖
npm install puppeteer

# 验证安装
node test-browser.js
```

### 2. 调用方式
```javascript
const AgentBrowser = require('./agent-browser');

async function yourWorkflow() {
    const browser = new AgentBrowser();
    await browser.init();
    
    try {
        // 你的业务逻辑
        const results = await browser.search('your keyword', 'baidu');
        // ...
    } finally {
        await browser.close();
    }
}
```

### 3. 配置建议
- 生产环境建议启用无头模式（默认）
- 高频爬取场景建议配置代理IP池
- 大型爬取任务建议复用浏览器实例
- 重要数据建议开启截图留存功能
