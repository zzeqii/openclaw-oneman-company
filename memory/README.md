# 记忆系统说明
本目录存储每日任务日志，属于临时数据，默认不会提交到Git仓库。

## 目录结构
```
memory/
├── YYYY-MM-DD.md       # 每日任务进展记录
├── enhancement/        # 记忆系统增强相关文件
└── README.md           # 本说明文件
```

## 记忆体系
1. **长时记忆**：根目录下的 `MEMORY.md`，存储永久核心规则、项目信息、关键决策
2. **当日日志**：`YYYY-MM-DD.md` 格式文件，每日自动生成，记录当天所有任务进展、变更记录
3. **实时追踪**：Paperclip常驻进程每5分钟自动同步所有变更，确保数据永不丢失

## 归档规则
- 每日日志保留90天，超过90天的日志自动归档到 `memory/archive/` 目录
- 重要项目信息会定期同步到长时记忆文件
