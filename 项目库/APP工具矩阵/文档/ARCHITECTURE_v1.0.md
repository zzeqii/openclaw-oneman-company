# APP工具矩阵技术架构设计
---
## 🎯 技术选型
采用 **Go后端 + Vue3前端** 技术栈，兼顾性能、开发效率和跨平台部署能力：
| 层级 | 技术选型 | 优势 |
|------|----------|------|
| 后端 | Go 1.22 + Gin框架 | 高性能、低内存占用、编译为单二进制文件，部署方便，适合工具类应用 |
| 前端 | Vue3 + Vite + Element Plus | 开发效率高、组件丰富、打包体积小，适合轻量工具类前端 |
| 存储 | SQLite（默认）/ MySQL（可选） | 轻量工具无需单独数据库，嵌入式SQLite足够，支持扩展为MySQL |
| 部署 | 单二进制文件+静态资源打包 | 整个应用编译为一个exe/dmg/linux二进制，用户双击即可运行，无需环境配置 |

## 🔧 五款工具规划
### 1. AI思维导图工具
**功能**：输入主题自动生成结构化思维导图，支持自由编辑、节点拖拽、多种布局（逻辑图/树状图/鱼骨图/组织结构图）、导出PNG/PDF/Markdown/Xmind格式
**技术实现**：
- 后端对接大模型生成结构化节点关系
- 前端用AntV X6可视化编辑，支持拖拽调整
- 支持多人协作、历史版本回溯
### 2. AI简历生成工具
**功能**：输入个人背景/经历自动生成专业简历，支持多套模板、岗位匹配优化、HR关键词优化、一页纸排版、导出PDF/Word格式
**技术实现**：
- 后端内置各行业简历模板+岗位关键词库
- 前端支持所见即所得编辑，实时预览效果
- 支持简历评分、优化建议生成
### 3. AI查重工具
**功能**：支持文档/论文/文案/代码查重，支持本地文件上传、粘贴文本查重、多语言支持、生成详细查重报告、重复率计算、相似片段高亮
**技术实现**：
- 后端本地构建分词+相似性匹配引擎，支持海量文本比对
- 前端支持批量上传，实时展示查重进度
- 支持导出详细查重报告，标注重复来源
### 4. 定时喝水提醒工具
**功能**：自定义喝水间隔、目标饮水量、提醒方式（弹窗/声音/通知栏）、喝水记录统计、周/月饮水量报表、缺水提醒、个性化饮水计划制定
**技术实现**：
- 后端本地定时任务，无需网络
- 前端提供可视化统计图表，打卡记录
- 支持多系统通知栏适配，不占用后台资源
### 5. 心情监控工具
**功能**：每日心情打卡、情绪记录、心情趋势分析、压力值评估、自动生成情绪报告、个性化情绪调节建议、冥想引导
**技术实现**：
- 后端本地存储所有数据，不上传到云端，保障隐私
- 前端提供心情日历、趋势图表可视化
- 支持自定义情绪标签、记录心情日记

## 📦 部署优势
- **跨平台**：支持Windows/macOS/Linux三端编译，用户无需安装任何依赖，双击即可运行
- **轻量**：整体体积<100MB，内存占用<50MB，启动速度<1秒
- **安全**：所有数据本地存储，不上传到云端，保障用户隐私
- **易扩展**：模块化设计，后续新增工具只需添加对应模块即可

## 🔍 选型对比（为什么不用Java）
| 维度 | Go | Java |
|------|----|------|
| 二进制体积 | <100MB | >500MB（含JRE） |
| 内存占用 | <50MB | >200MB |
| 启动速度 | <1秒 | >5秒 |
| 部署复杂度 | 单文件直接运行 | 需要安装JRE，配置环境变量 |
| 开发效率 | 语法简洁，标准库强大 | 生态重，配置多 |
工具类应用优先选择Go栈，性能更好、部署更简单、用户使用成本更低。如果后续有企业级复杂业务场景，可以切换为Java SpringBoot架构。
