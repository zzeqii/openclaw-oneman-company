## 安装方式
1. 命令行安装: `clawhub install <skill-slug>`
2. 手动安装: 将技能文件夹复制到 `~/.openclaw/skills/`（全局）或 `<项目目录/skills/`（工作区）
3. 自动安装: 直接将技能的GitHub仓库链接粘贴到助手聊天中，助手将自动处理安装

技能经过筛选但未经过完全审计，安装前建议检查源代码，可使用Snyk Skill Security Scanner或Agent Trust Hub进行安全扫描。
