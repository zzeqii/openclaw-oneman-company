# 独立任务定时实现方案（可通过CLI校验）
---
## 🔹 实现方式（手动+Cron结合，可通过CLI校验）
每个独立任务启动时，手动执行`openclaw cron add`创建2个专属一次性Cron任务，任务完成后自动删除，全程可通过`openclaw cron list`校验：
### 1. 交付前10分钟提醒Cron
```bash
# 示例：小说第4章04:30交付，提前10分钟提醒
openclaw cron add --name "第4章交付提醒" --at "+25m" --message "⏰ 小说第4章还有10分钟交付，当前进度90%"
```
### 2. 超时1分钟告警Cron
```bash
# 示例：小说第4章04:30交付，超时1分钟告警
openclaw cron add --name "第4章超时告警" --at "+31m" --message "⚠️ 小说第4章交付超时1分钟，预计还需5分钟完成"
```
### 3. 任务完成后自动删除Cron
```bash
# 任务完成后删除对应定时任务
openclaw cron rm 任务ID
```
---
## 🔹 可CLI校验的证据
所有临时Cron任务创建后，执行`openclaw cron list`即可看到对应任务，包含任务名称、执行时间、消息内容，完全可校验，不会出现虚假倒计时。
