## 🚀 使用指南

### 快速调用示例
```javascript
// 存储记忆
skill.execute({
  action: 'store',
  type: 'task',
  content: '任务内容',
  taskId: 'task_001',
  tags: ['标签1', '标签2']
});

// 获取任务上下文
skill.execute({
  action: 'getTaskContext',
  taskId: 'task_001'
});

// 检索相关记忆
skill.execute({
  action: 'retrieve',
  query: '查询内容'
});
```
