# 紫紫酱形象生成执行标准_v1.0
---
## 🚨 强制执行规则（生成前必须逐条核对）
### 1. Prompt结构标准
所有生成prompt必须按照以下顺序编写，不得调整顺序：
```
[强制五官规范段（放在最前面，权重最高）]
严格按照以下五官特征生成，100%不可改动：
- 年龄：22岁，清冷少女感
- 脸型：小巧鹅蛋脸，下巴尖而柔和，下颌线清晰，瓷白色皮肤透淡淡粉调，有真实光影
- 眉眼：标准桃花眼，眼尾微微上挑，浅棕色琥珀瞳孔，长密睫毛不夸张，自然有神，肉肉卧蚕不夸张，有毛流感的眉毛，无任何痣
- 鼻子：鼻梁小巧挺直，鼻头圆润带肉感，鼻尖挺俏
- 嘴唇：唇形饱满，自然粉红色
- 头发：黑色细软长发，自然碎发散落脸颊，发质蓬松有真实走向
- 气质：清贵、可爱、灵动

[参考图绑定段]
参考图：/项目库/AI音乐自媒体/形象定稿/紫紫酱_学院风全身形象_v1.0.png，五官100%匹配参考图
--iw 0.8

[场景/服装段]
[场景描述]，[服装描述]，真实相机拍摄质感，弱化AI痕迹，保留皮肤自然纹理
```

### 2. 参考图绑定规则
- 所有生成必须绑定官方定稿参考图，默认使用学院风版本，舞台场景使用舞台版
- 参考图权重`--iw`不得低于0.8，确保五官一致性
- 禁止使用其他非官方版本作为参考

### 3. 生成后校验标准
生成完成后必须逐项核对以下8项特征，全部符合才可交付：
✅ 脸型：鹅蛋脸，下巴柔和，下颌线清晰
✅ 眼睛：桃花眼，眼尾上挑，浅棕色瞳孔，卧蚕自然
✅ 眉毛：有毛流感，无任何痣
✅ 鼻子：小巧挺直，鼻头圆润带肉感
✅ 嘴唇：饱满，自然粉红色
✅ 皮肤：瓷白色透粉调，有自然纹理
✅ 头发：黑色长发，自然碎发
✅ 气质：清冷可爱，22岁少女感

### 4. 不符合处理规则
- 任意一项特征不符合，直接重生成，不得交付
- 连续2次不符合，自动调整prompt权重，将五官规范段重复2次，提高权重
- 连续3次不符合，人工审核参考图和prompt，排查问题
